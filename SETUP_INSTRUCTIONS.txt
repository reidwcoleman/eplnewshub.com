1. Create Supabase Account (FREE)

2. Create the Database Table
   - In Supabase dashboard, go to "Table Editor"
   - Click "New Table"
   - Table name: leaderboard
   - Add these columns:
     * user_id (text, primary key) ‚Üê CHECK THE PRIMARY KEY BOX
     * username (text)
     * total_points (int8, default: 0)
     * streak (int8, default: 0)
     * total_predictions (int8, default: 0)
     * correct_predictions (int8, default: 0)
     * accuracy (int8, default: 0)
     * last_updated (timestamptz, default: now())
   - Click Save

3. Enable Public Access (IMPORTANT!)
   - Go to "Authentication" ‚Üí "Policies"
   - For the "leaderboard" table:

   CREATE POLICY 1:
   - Name: Enable read access for all users
   - Policy: SELECT
   - Target roles: public
   - USING expression: true

   CREATE POLICY 2:
   - Name: Enable insert/update for all users
   - Policy: INSERT, UPDATE
   - Target roles: public
   - WITH CHECK expression: true

4. Get Your API Credentials
   - Go to "Settings" ‚Üí "API"
   - Copy TWO things:
     a) Project URL (looks like: https://xxxxx.supabase.co)
     b) anon public key (long string starting with eyJ...)

5. Update epl-predictor-game.html
   - Open the file
   - Find lines 1352-1353 (near the top of the <script> section)
