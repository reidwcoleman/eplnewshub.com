FPL Banner Popup Test Page

This page is for testing the FPL tools advertisement banner popup.

Status: The popup should appear 3 seconds after page load.
Features:
  • Full-screen overlay with blur effect
  • Animated entrance (slide up + fade in)
  • Gradient background based on ad priority
  • Close button with rotation animation on hover
  • "Maybe Later" secondary CTA
  • Cookie-based dismissal (won't show again for 24 hours)
  • Responsive design for mobile
