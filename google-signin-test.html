üìã Firebase Setup Requirements

To make Google Sign-In work, you need to configure Firebase properly:

  1. Go to Firebase Console: Open Firebase Console
  2. Enable Google Provider:
    • Click on "Google" in the Sign-in providers list
    • Toggle the "Enable" switch
    • Add a support email (required)
    • Click "Save"
  3. Add Authorized Domains:
    • Go to Authentication ‚Üí Settings
    • Add your domain to "Authorized domains"
    • For local testing, localhost is already authorized
    • For production, add: eplnewshub.com
  4. Configure OAuth Consent Screen:
    • May need to be done in Google Cloud Console
    • Set app name, support email, and authorized domains
